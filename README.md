## Employee Workforce Analytics (Excel Project)
## Project Overview

- This project analyzes employee data to understand workforce demographics, salary distribution, and performance trends using Excel.

- It helps HR teams make data-driven decisions for better workforce planning and organizational growth.

## Objectives
- Analyze employee demographics
- Evaluate department-wise performance
- Study salary distribution patterns
- Identify workforce trends
- Support HR decision-making
## Dataset Information

The dataset contains employee records including:

- Employee ID
- Name
- Department
- Country
- Salary
- Date of Joining
- Performance Rating
- Gender
- Email
## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Data Cleaning Functions

## Key Insights
- Sales department has highest salaries
- Marketing has highest performance ratings
- Most employees are from India
- Hiring trend is increasing yearly
- Salary impacts performance in some departments
##  Business Recommendations
- Improve Finance department incentives
- Increase salary in Marketing for better performance
- Strengthen Sales team for revenue growth
- Focus on employee retention strategies
##  Conclusion

This project demonstrates how Excel can be used for HR analytics to understand workforce behavior and support strategic decision-making.
